# Plans

## Overview

a command line tool to generate project templates and/or boilerplate

## Why

I'm making this because it would be great practice for learning C for uni, and because im sick of writing boiler plate for the projects I start.

## Features

- optional boilerplate
- multiple project languages supported

## development dependancies

- NCurses

## usage

```shell
    mp <name>
```

- mp: the application name
- <name>: your project's name
